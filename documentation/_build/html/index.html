Welcome to the AeroQuad Configurator Developer Guide’s documentation!¶

This is the developer’s guide for the Python based AeroQuad Configurator. The Configurator is a visual tool used to verify the correct setup and operation of your AeroQuad flight control board. It can also upload the latest firmware to take advantage of the latest flight control features.

This developer’s guide will show you how to add your own features and capabilities to the AeroQuad Configurator.

If you are looking for a User’s Guide for the Configurator, please check out our wiki!

Installation Requirements¶

  • Python v2.7
  • PyQt with Qt Designer (get Binary Package)
  • Eclipse (you’ll download PyDev inside Eclipse, I got the C++ version)
  • PyDev (can execute Python code directly from Eclipse)
  • Setup code completion in Eclipse for PyDev
  • PySerial (serial drivers for Python)
  • PyQtGraph
  • NumPy
  • SciPy
  • How to setup Git inside of Eclipse
  • The GitHub repo is at: https://github.com/AeroQuad/AeroQuadConfiguratorPyQt.git

Configuring Eclipse:¶

  • Insure c:\Python27 is in PATH
  • Change PyDev - Interpreter from default to python
  • Update PyDev - PYTHONPATH -> External Libraries -> pyqtgraph to point to python27
  • If you want to hide certain files/folders in the Eclipse project view
  • To run the application, look for AeroQuadConfigurator.py in the project window and double click it to open the file. Then hit the Run button (the green circle with a triangle button pointing to the right) to execute it.
